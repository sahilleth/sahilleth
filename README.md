## 🔬 Research

I work on cross-lingual transfer and LLM evaluation across 
non-Latin scripts. My current research investigates how model 
scale and prompting strategy affect performance across diverse 
script families.

**Current paper:** Scale Helps, Few-Shot Hurts: Evaluating LLMs 
Across Five Non-Latin Scripts

Key findings:
- Scaling from 8B to 70B closes the English-bias for Hindi, 
  Thai and Chinese (+8%) but Arabic uniquely resists (+2%)
- Few-shot prompting hurts non-Latin scripts by 3–6% while 
  helping English — Arabic suffers most (−6%)
- Tokenizer inflation alone does not explain Arabic's 
  resistance — morphological complexity is the key factor
